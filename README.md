# Gamma - Excel Record Validator

Desktop app for reviewing Excel records and detecting validation errors based on incidence rules.

## Quick start
- Create a virtual environment (optional)
- Install requirements
- Run the app

Example:
```
python -m pip install -r requirements.txt
python -m app.main
```
